# React Tic-Tac-Toe

一個使用 React 開發的井字棋遊戲，支援雙人對戰、動態顯示當前玩家、勝負判定以及重新開始功能。

## 🕹 遊戲特色

- 支援玩家改名
- 回合紀錄 Log
- 自動判斷勝負與平手
- 一鍵重新開始遊戲
